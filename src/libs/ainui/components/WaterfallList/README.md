# WaterfallImages

> 瀑布流图片布局

## 特征

 - 图片宽度固定，高度不固定
 
 - 图片从左到右排列，一行排满后，其余图片按顺序排在短的一列后

## 思路
  - 定义保存图片高度数组 `arr`
  - 循环数据，保存加载完成后图片的高度
  - 找到 `arr` 数据中最小高度的图片
  - 设置对应的位置
